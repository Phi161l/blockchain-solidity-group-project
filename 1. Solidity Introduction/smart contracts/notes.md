# 🧠 Smart Contracts — Notes

## Course Overview

* Learn Solidity basics
* Understand how smart contracts work
* Write simple blockchain programs

---

## Smart Contracts Overview

* Smart contracts are programs stored on a blockchain
* They run automatically when conditions are met
* No middleman (code enforces rules)
* Used for payments, apps (DApps), tokens

---

## Value Types

* Store simple data directly
* Fixed size, cheap to use

Examples:

* `bool` → true/false
* `uint` → unsigned numbers
* `int` → signed numbers
* `address` → wallet address
* `bytes32` → fixed-size data

---

## Practice Data Types

* Used different Solidity types in examples:

  * Booleans (`true/false`)
  * Unsigned integers (`uint8`, `uint16`)
  * Signed integers (`int8`)
  * Strings and bytes
  * Enums (custom types)

---

## One-line Summary

Smart contracts = programs on blockchain that automatically enforce rules using data types like bool, int, string.
